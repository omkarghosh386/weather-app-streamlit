# 🌤️ Weather App (Streamlit)

A simple and elegant weather web app built using **Streamlit** and the **OpenWeather API**.

## 🚀 Features
- Live weather data by city 🌍  
- 3-day forecast 📅  
- Dynamic background color 🎨  
- Celsius/Fahrenheit toggle 🌡️  
- Fully responsive UI  

## 🧩 Tech Stack
- Python
- Streamlit
- OpenWeatherMap API
